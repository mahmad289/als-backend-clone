To run the CMD 

npx nestjs-command `command name`

//Install Commands

- To install tags data
npx nestjs-command run:install-tags-manager-data

- To install master data
npx nestjs-command run:install-master-data

- To install uat master data
npx nestjs-command run:install-uat-master-data

- To install master requirement data
npx nestjs-command run:install-master-requirement-data

- To install template data
npx nestjs-command run:install-template

- To install requirement group data
npx nestjs-command run:install-requirements

- To install project data
npx nestjs-command run:install-project-data

-To install the communication template data
npx nestjs-command run:install-communication-template-data

- To install file manager data
npx nestjs-command run:install-file-manager-data

- To install the auto notification data
npx nestjs-command run:install-auto-notification-data

- To install complete database
npx nestjs-command run:install-complete-data
